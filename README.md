### Servlet의 개요  
웹 서버상에서 실행되는 자바의 클래스 파일  
javax.servlet.Servlet 인터페이스를 구현(Implements)해서 작성  
입력과 출력을 HTTP 프로토콜 요청(Request)과 응답(Response)의 형태로 다룬다
